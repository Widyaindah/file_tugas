Hei semua!

Saya ingin perkenalkan tentang diri saya dulu, nama saya Widya Indah Permatasari, bisa dipanggil Widya, saya berasal dari Bekasi/Indonesia, saya lahir pada tanggal 11 Februari 2001. Sampai sekarang saya masih menetap di Bekasi, Saya akan membahas pendidikan saya, sekarang saya berstatus mahasiswa di salah satu kampus Swasta yaitu Universitas Esa Unggul, Jurusan Teknik Informatika.

Sekarang saya akan membahas tentang hobi, hobi saya mungkin banyak, namun hobi yang paling saya sukai yaitu hobi bidang photografi, bahkan sekarang hobi saya ini sekaligus menjadi pekerjaan saya dalam mendapat uang, untuk bidang fotografi sendiri saya sudah berkakrir kurang lebih selama 4 tahun, disitu banyak sekali saya mendapatkan pengalaman besrta pengetahuan, sehingga membuat saya makin tertariik untuk berkarir dibidang fotografi, bahkan sampai sekarang aktifitas saya dipenuhi dengan memotret. Dibalik memotret saya juga suka mengolah foto dengan memanipulasi beberapa gambar menjadi satu gambar, dan membuatnya menjadi lebih real, atau biasa juga disebut digital imaging. Bahkan sampai sekarang trick memanipulasi ini menjadi sebuah trick yang sangat handal. Selain hobi fotografi saya juga hobi adventure, bagi saya menjelajah itu sangat keren dan bisa membuat otak dan pikiran saya segar kembali, karna pada saat saya berada di meja kerja itu membuat pikiran saya jenuh, makanya dengan itu saya kadang-kadang memanfaatkan waktu saya untuk berpetualang dengan kawan-kawan saya. Disaat saya melihat pemandangan alam yang begitu indah itu membuat saya sangat terkagum, melihat anugerah yang tuhan ciptakan dengan likak likuk gunung beserta air yang sangat tenang kadang membuat ide-ide kreatif saya terlahir, dari ide kreatif itu saya manfaatkan untuk membuat karya dibidang fotografi saya makin kreatif.
